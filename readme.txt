== Installation ==

### Install Theme using WordPress Theme Search

1. Go to Dashboard Admin Panel >> Appearance >> Themes >> Add New.
2. Search for 'iCare Fitness'.
3. Click **Install Now** button.
4. Then click **Activate Theme** button.
5. Now theme is Activated.

### Install Theme using the WordPress Admin Theme Upload

1. First download the 'iCare Fitness' theme ZIP package from WordPress.org.
2. Log in to your website administrator panel.
3. Go to Appearance >> Themes >> Add New & click on “Upload Theme”.
4. Upload [iCare Fitness](https://wordpress.org/themes/icarefitness/) ZIP file by choosing it from your computer.
5. Click **Install Now** button.
6. Then click **Activate Theme** button.

### Using FTP

1. First you will need to download the iCare Fitness theme ZIP package from WordPress.org.
2. Next, you need to extract the zip file on your computer.
3. Login to your FTP.
4. Access the path /wp-content/themes/ for your server directory.
3. Upload the extracted folder to the **/wp-content/themes/** folder on your web server.
4. Activate the theme from WordPress Admin Panel.
